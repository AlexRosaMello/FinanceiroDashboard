# FinanceiroDashboard

Este projeto é um sistema de dashboard financeiro desenvolvido em .NET, SQLServer, HTML, CSS, jQuery, Boostrap e Razor. Ele permite a visualização e gestão de notas fiscais, com funcionalidades como filtros avançados, exibição de totais e gráficos dinâmicos.

## Funcionalidades

- Listagem de Notas Fiscais com filtros por data e status
- Dashboard com resumo financeiro:
  - Valor total das notas emitidas
  - Valor total das notas emitidas sem cobrança realizada
  - Valor total das notas vencidas
  - Valor total das notas a vencer
  - Valor total das notas pagas
- Gráficos gerados com Chart.js
- Conexão com banco de dados SQL Server
- Implementação com Docker e IIS Express

## Tecnologias Utilizadas

- **Ferramenta:** Visual Studio 2022 - Community
- **Backend:** .NET 8, C#
- **Frontend:** HTML5, CSS3, Jquery v3.7.1, Razor, Bootstrap v5.3.3
- **Banco de Dados:** SQL Server Express
- **Deploy:** Docker, IIS Express
